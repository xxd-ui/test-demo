Day 01

HTML 骨骼架式

html 标签:网页的根节点
head 标签: 辅助浏览器解析页面，并不会在页面中展示。
body 标签：页面的主体部分，用于存放网页要展示的标签。

HTML 常用标签：

注释标签：<!-- 注释语句 -->
文档类型 <!DOCTYPE> ：声明文档类型，告诉浏览器以 HTML5 的标准去解析页面。
<meta> 标签：设置网页的元数据，不同的属性会使<meta>标签具备不同的功能。
   1. <meta charset="utf-8" />指定字符集编码 utf-8 (优化后的全球码)
   2. 用于设置关键字 <meta name="keywords" content="xxx" />
   3. 用于设置描述信息 <meta name="description" content="xxxx" />
标题标签：<hn>标题文本</hn> h1 最大 h6 最小
段落标签:<p>文本内容<p>
分割线标签:<hr/>
换行标签:<br/>
斜体 <em> <i>
加粗 <strong><b>
上标/下标 <sup><sub>
下划线/删除线<ins><del>
转义字符 空格 &nbsp ; 小于号&lt ; 大于号&gt ;
图像标签: <img src="" alt="">
   src 属性用于指定图像文件的路径和文件名.
   alt 属性用于指定图片未找到时，显示的内容.

路径(相对路径和绝对路径)

相对路径:相对于当前文件
   同一级目录 直接引用图像文件的名称
   上一级目录 在文件名之前加../ ，如果上两级目录，使用 ../ ../
   下一级目录 输入文件夹名和文件名，之间用/隔开
绝对路径:
本地绝对路径（不用）
网络路径（常用）

链接标签

<a href="跳转目标" target="目标窗口的弹出方式">文本或图像</a>
   href：用于指定链接目标的 url 地址.
   target：用于指定链接页面的打开方式。
      _self 为在当前窗口打开（默认值）。
      _blank 为在新窗口中打开方式。
      _top 在顶级窗口打开。（不常用）
      _parent 在父类窗口打开。（不常用）

HTML基本规范(语法规范)

注释不能嵌套
标签必须完整
标签可以嵌套，但要注意语义。
标签的属性必须加双引号
标签属性尽量使用小写

表格标签

	     <th>表头内的文字</th>
	     ...
	 </tr>
	 <tr>
	    <td>单元格内的文字</td>
		...
	 </tr>
	...
</table>
